# Codex Dream Skin · Windows

## 已安装旧版本的用户

- 如果此前安装的是 `windows-v1.3.3` 或 `windows-v1.4.0` ZIP，请下载本次 `Setup.exe` 并运行一次；不需要卸载旧版，也不需要先恢复官方主题。
- 安装器会关闭 Dream Skin 托盘和 Codex，原子替换运行时并重建快捷方式。
- 当前主题、已保存主题和导入图片默认保留。
- 从本版本开始，托盘每天最多检查一次稳定版更新，也可以随时点击“检查更新”。更新清单与安装包都由 GitHub Release 托管，安装包必须通过清单中的 SHA-256 校验后才会启动。

## 全新安装

下载 `Codex-Dream-Skin-Windows-*-Setup.exe` 后双击运行。安装器按当前用户安装，不要求管理员权限。

## 卸载

在 Windows“已安装的应用”或开始菜单中卸载 `Codex Dream Skin`。卸载时可以选择保留主题数据，方便以后重新安装，也可以一并清除。

## 安全声明

非 OpenAI 官方产品。主题通过本机回环 CDP 加载，不修改 WindowsApps、`app.asar` 或官方代码签名。未签名构建可能触发 Windows SmartScreen；正式分发建议配置代码签名证书。
